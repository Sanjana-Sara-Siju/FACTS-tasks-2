# Fabric Similarity & Recommendation Task (Backend)
**Client Context:** Al Kilani Fabrics

## Overview
This repository contains a **Hybrid Content-Based Image Retrieval (CBIR) System** designed to automate fabric substitute recommendations. When a specific fabric is out of stock, this pipeline evaluates alternative fabrics to ensure they meet a strict 70% visual similarity threshold.

This system separates the analysis into mathematical channels (pure color and pure texture) and  semantic structures, ending with a cognitive report generated by Mistral AI.

## Key Features
* Utilizes a lightweight Hugging Face Vision Transformer (`google/vit-base-patch16-224`) to calculate the overall macro-structure and layout similarity via cosine similarity.
* Extracts and compares normalized 2D HSV Histograms using OpenCV, separating color tint (Hue) and intensity (Saturation) from lighting variables (Value).
* Applies Local Binary Patterns (LBP) via scikit-image on grayscale conversions to mathematically compare the physical ridges, thread density, and weave architecture of the fabrics.
* Integrates Mistral AI (pixtral-large-latest) to analyze both the raw images and the calculated mathematical metrics, generating a human-readable justification for whether a fabric is an acceptable substitute.

## Installation and Setup
1. **Clone the repository and navigate to the task directory**:

git clone [https://github.com/Sanjana-Sara-Siju/FACTS-tasks-2.git](https://github.com/Sanjana-Sara-Siju/FACTS-tasks-2.git)
cd FACTS-tasks-2/image-comparison-task


2. **Install required dependencies**:

Ensure you have Python 3.8+ installed, then run:

pip install opencv-python scikit-image scikit-learn transformers torch pillow mistralai python-dotenv

3. **Configure Environment Variables**:

Create a .env file in the image-comparison-task directory and add your Mistral API key:

MISTRAL_API_KEY=your_actual_api_key_here

## Usage
The primary execution script is main3.py, which runs the full hybrid pipeline.

1. Ensure your 2 comparison images are in the directory and correctly referenced in the code (e.g., img1_path = "fabric3.png").

2. Execute the script: python main3.py

## The Scoring algorithm
The final compatibility score is calculated using a weighted ensemble method to prevent single-attribute false positives (e.g., two fabrics being the exact same colour but entirely different materials):

40% - Semantic Structure (ViT Embedding Cosine Similarity)

30% - Color Similarity (HSV Histogram Correlation)

30% - Texture Similarity (LBP Array Cosine Similarity)

## Model Architecture and Libraries
Computer Vision: OpenCV (cv2), scikit-image (skimage)

Deep Learning (Embeddings): PyTorch (torch), Hugging Face Transformers

Array Math & Similarity: NumPy, scikit-learn (cosine_similarity)

LLM: Mistral AI Python SDK
